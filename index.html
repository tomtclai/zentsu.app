We build small, focused software for Apple platforms.

Zentsu LLC is a software studio working on native tools and apps for macOS and iOS. We favor private-by-default design, one-time purchases where the model fits, and shipping software that does a clear job without asking for an account.

{% include apps-section.html %}

How we build

  • Native first

    Mac and iOS apps written in Swift and SwiftUI, not cross-platform shells wrapped around a web view.

  • Private by default

    Our apps work without a Zentsu account or telemetry. When an app syncs, it uses your private iCloud storage instead of a Zentsu server.

  • Pay once

    One-time purchases over subscriptions, where the model fits. The version you bought keeps working.
