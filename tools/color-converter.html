Color Converter for Mac

A native Mac color converter. Paste a HEX, RGB, HSL, or OKLCH value and get every other format back at once. No conversion chart, no browser required.

What it does

Bench's Color Converter takes a color value in any supported format — #3B82F6, rgb(59, 130, 246), hsl(217, 91%, 60%), or oklch(0.63 0.19 256) — and outputs all the others simultaneously, with a live color preview swatch. Switch between input formats with a single click. The OKLCH output is particularly useful for modern CSS work where perceptually uniform color spaces are more predictable than RGB or HSL when adjusting lightness.

Why use Bench's color converter

Design tokens in one format often need to feed CSS in another. Figma outputs HEX; your design system uses OKLCH; your Tailwind config wants HSL. Web converters handle the conversion but add a browser tab, a page load, and usually some ads. Bench's converter runs locally in a single SwiftUI window — paste the value, copy what you need, and get back to work. No data leaves your Mac.
