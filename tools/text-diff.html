A native Mac text diff tool. Two chunks of text, side by side. You'll see exactly what changed: added lines in green, removed lines in red, unchanged lines in context.

What it does

Bench's Text Diff tool takes two text inputs and produces a side-by-side line diff. It highlights added, removed, and modified lines so you can quickly find what changed between two versions of a config file, a log snippet, a JSON response, or any other text. The diff runs locally using a standard longest-common-subsequence algorithm. No server involved, no copy of your text sent anywhere.

Why use Bench's text diff tool

Web-based diff tools require you to paste both text blocks into a form on a remote server. That's fine for public content, but config files, environment variable dumps, and API response payloads frequently contain credentials or internal identifiers you'd rather not share. Bench keeps both inputs local on your Mac, in a sandboxed app with no network entitlement.

  • Native Mac app. SwiftUI, single window, instant launch.
  • Offline. No network calls. Whatever you paste stays on your Mac.
  • No tracking. Nothing you enter is logged, sent, or shared. Bench has no analytics and no third-party SDKs.
