What it does

Bench's Markdown preview takes CommonMark-compatible Markdown and renders it to styled HTML in a side panel. Headings, bold, italic, inline code, fenced code blocks, tables, and blockquotes all render correctly. The preview updates as you edit the source pane, so you see the result without switching contexts. It's useful for READMEs, pull-request descriptions, changelogs, and any other structured text you're writing in Markdown before it lands somewhere public.

Why use Bench's Markdown preview

Most editors have Markdown preview built in, but it's another setting to find and another mode to toggle. Bench is one keystroke away from any app — paste the Markdown, check the render, copy the result or go back to your editor. For quick spot-checks, that's faster than configuring a preview panel in a tool you don't normally use for writing.
