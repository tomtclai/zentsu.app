Image Diff for Mac

A native Mac image diff tool. Drop two PNG files onto Bench and the pixels that moved light up. Useful for screenshot testing, UI regression checks, and design reviews.

What it does

Bench's Image Diff tool compares two PNG images and produces a diff overlay where changed pixels are highlighted. You can see the before and after side by side, or view the composite diff image that shows only the regions that changed. It's built for the common case in front-end and iOS development: you took a screenshot before and after a change and want to confirm what moved. The comparison runs entirely on your Mac. Neither image is uploaded anywhere.

Why use Bench's image diff tool

Screenshot comparison is a common step in UI review and snapshot testing. Web-based diff tools require uploading both images to a third-party server, which is fine for public screenshots but not for internal dashboards, analytics views, or anything showing real user data in the UI. Bench keeps both files local. Drop them in, get the diff, close the tool.
