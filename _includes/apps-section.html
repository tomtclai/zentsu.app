{%- comment -%} apps-section.html: the shared "Apps" band, rendered from _data/apps.yml. This is the ONE copy of the studio's app listing. It is included by BOTH the studio homepage (index.html) and the standalone /apps page (apps.html) so the two never drift. To change how apps are listed, edit here once. Apps with an explicit `visible: false` are removed first; missing visibility values are visible by default. Live apps then render before coming-soon, while `parked` apps are omitted. To add an app: one entry in _data/apps.yml plus an icon, with no markup change here. See _data/apps.yml for the full "how to add an app" notes and the one-line visibility switch, and _includes/app-card.html for the per-card markup. {%- endcomment -%}
